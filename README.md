#Skidmatchning
Ett program som används för att matcha skidskoleelever med skidor, pjäxor och stavar.

## Skidor
1. Ladda ner exempelfil för skidor från https://andersper67.github.io/Skidmatchning_public/skidor.csv
2. Inventera och uppdatera filen
3. Ladda in filen i programmet

## Pjäxor
1. Ladda ner exempelfil för pjäxor från https://andersper67.github.io/Skidmatchning_public/pjäxor.csv
2. Inventera och uppdatera filen
3. Ladda in filen i programmet

## Stavar
1. Ladda ner exempelfil för stavar från https://andersper67.github.io/Skidmatchning_public/stavar.csv
2. Inventera och uppdatera filen
3. Ladda in filen i programmet

## Listan med Anmälda 
1. Logga in till https://idrott.se 
2. Navigera till Admin -> Övr -> Frågeformulär.
3. Klicka på  "Skidskolan anmälan 2026" (men rätt år ;-)) och klicka på "Visa inkomna svar" (uppe till höger) 
4. Klicka på "Exportera sökresultatet" (ett litet moln med Nedåtpil, man kan få klicka på expandera ...) och sedan CSV för att ladda ner filen.
5. Nu måste man dela upp filen på så många dagar som man har skidskola. Tex har man tisdag och torsdag så måste man välja ut de elever som kör på respektive dag och skapa en separat file för respektive dag.

## Matcha
1. Ladda upp filen med Anmälda för aktuell dag (enl ovan)
2. Klicka på "Kör Matchning"
3. Kontrollera efter [ERROR] eller [WARNING] längst ned på sidan i loggen. Tex kan programmet inte matcha de som inte anget all info så lite manuellt jobb krävs!
4. Repetera för andra dagar.
   



Alla uppladdade filer sparas i "localstorage" för webbläsaren och man behöver normalt bara ladda upp skidor, pjäxor och stavar en gång
